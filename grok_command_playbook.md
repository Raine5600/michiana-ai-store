# Grok Command Playbook — Michiana AI Solutions
**Copy any command below into Grok. I execute the work. You paste/send where noted.**

---

## How This Works

| You do (minutes) | Grok does (everything else) |
|------------------|----------------------------|
| Paste copy into ConvertKit / Gumroad / Fiverr | Write all copy, emails, posts, proposals |
| Hit Send on outreach emails | Personalize 10–20 emails per batch |
| Post to FB groups (copy-paste) | Generate posts + variations |
| Take discovery calls (optional) | Prep scripts, objection handlers, follow-ups |
| Connect Zapier once | Document exact zap steps |

**Daily routine:** Open this file → run today's Grok commands → paste outputs → done.


```
/setup-gumroad
Write Gumroad product descriptions for all 4 kits ($47, $67, $97, $147 bundle). Include tags, cover copy, and what files to attach from my Downloads folder.
```

```

```

```

List exact Zapier zaps I need (trigger + action steps) for Gumroad sales logging and ConvertKit lead alerts. One zap per numbered step.
```

## Daily Sales Commands (Service — $499–$999)

### Find targets
```
/find-prospects [trade] [city]
Search for 15 [plumbers/HVAC/dentists] in [South Bend/Niles/Granger] with outdated or no website. Output a table: business name, website URL, gap (no site / old site / no chat), suggested pitch angle.
```

### Personalized outreach
```
/write-outreach
For [Business Name] — [trade] in [city]. Website: [URL or "none"]. Write a cold email under 120 words, subject line, and a 2-sentence FB DM version. Mention our live plumber demo. Sign as Cameron, Michiana AI Solutions, 574.274.0643.
```

### Batch outreach (fastest)
```
/batch-outreach [trade] [count]
Write [10] personalized cold emails for [HVAC companies] in Michiana. Vary subject lines. Each email must reference something specific (old WordPress site, no mobile, no online booking). Include demo link placeholder [DEMO_URL].
```

### Follow-up
```
/follow-up [days]
Write follow-up #[2] for businesses that didn't reply to my AI website + chatbot pitch. 3 versions: friendly bump, value-add (share stat), breakup email. Under 80 words each.
```

### Discovery call prep
```
/prep-call [Business Name] [trade]
Create a 1-page call sheet: 5 discovery questions, demo flow (which pages to show), recommended package ($499/$799/$999), objection handlers, and close script. Reference sample_plumber_website_demo.html.
```

### Proposal
```
/proposal [Business Name] [trade] [package]
Write a 1-page client proposal: scope, timeline (48hr), deliverables (site pages, chatbot features), price, payment terms (50% upfront), next step. Professional but not corporate.
```

### Post-call
```
/post-call [outcome]
They [said yes / want to think / said too expensive]. Write the exact email to send within 1 hour, plus calendar invite text if they said yes.
```
