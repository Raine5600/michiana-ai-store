# Passive Weekly Playbook — 1 Hour Per Week
**No outreach. No calls. Grok generates; you paste.**

---

## Monday (45 min) — Content batch

Paste into Grok:
```
/passive-content-week
Generate for Michiana AI digital products ($47-$147):
- 5 Pinterest pin titles + descriptions + image prompts
- 1 Reddit value post for r/sidehustle (not salesy)
- 1 LinkedIn post (digital product angle)
- 1 X/Twitter thread (5 tweets) about starting AI agency with kits
CTA: free checklist on storefront. Products: chatbot prompts, launch kit, bundle.
```

**You do:**
- [ ] Schedule pins in Pinterest (or Tailwind)
- [ ] Post Reddit if rules allow (value-first)
- [ ] Schedule LinkedIn + X (Buffer free tier or native)

---

## Wednesday (5 min) — Check numbers

- [ ] Gumroad dashboard: sales this week
- [ ] ConvertKit: new subscribers, sequence completion rate
- [ ] Google Sheet (Zapier): any anomalies

If sale came in: celebrate. No action required (auto-delivered).

If support email: paste into Grok:
```
/passive-support [paste buyer message]
```

---

## Friday (10 min) — Optional single post

Pick ONE (not all):
- [ ] New Pinterest pin from batch
- [ ] Reply to any Reddit comments on your post
- [ ] Update one Gumroad tag/description if zero views

---

## Monthly (30 min, first Monday of month)

Paste into Grok:
```
/passive-review
This month Gumroad: [X sales, $Y revenue, top product]
ConvertKit: [Z new subs]
Recommend: new SKU, pricing tweak, or traffic channel to double.
```

Optional:
```
/passive-blog [topic from review]
Write 800-word SEO article with meta description.
```

Publish to: Medium, storefront `/blog`, or LinkedIn article.

---

## Quarterly (1 hr)

- [ ] Zip updated kit files → upload new version to Gumroad ("Lifetime updates")
- [ ] Add one $19 niche prompt pack (Grok splits chatbot prompts by vertical)
- [ ] Review BUNDLE20 code performance
